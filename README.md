# 🌈 นิทานหรรษา — เว็บเล่านิทานสำหรับเด็ก

เว็บเล่านิทานที่ใช้ **เสียง AI ภาษาไทย** เล่าเรื่อง พร้อม **ภาพประกอบ** และให้เด็ก **แตะแต่ละฉากเพื่อฟัง** ได้เอง

## ✨ ฟีเจอร์
- 📖 นิทาน 9 เรื่อง พร้อมข้อคิดสอนใจ (กระต่ายกับเต่า, ลูกเป็ดขี้เหร่, ราชสีห์กับหนูน้อย, ลูกหมูสามตัว, เด็กเลี้ยงแกะ, มดกับตั๊กแตน, หมาจิ้งจอกกับองุ่น, กากับเหยือกน้ำ, สุนัขกับเงาในน้ำ)
- 🎧 เสียง AI เล่าภาษาไทยจาก **Microsoft Edge Neural TTS** (เสียง `th-TH-PremwadeeNeural`) สร้างเป็นไฟล์ MP3 ไว้ล่วงหน้า → เสียงธรรมชาติ เล่นได้ทันที ไม่ต้องต่อเน็ต ไม่ต้องใช้ API key
- ⚡ ปรับความเร็วได้ 3 ระดับ (🐢 ช้า / 🙂 ปกติ / 🐇 เร็ว) เปลี่ยนได้สดขณะฟัง โดยระดับเสียงไม่เพี้ยน
- 🎨 ภาพประกอบทุกฉากเป็น SVG วาดในโค้ด คมชัดทุกขนาดจอ
- 👆 เด็กแตะรูป/ปุ่มเพื่อฟัง หยุด/เล่นต่อได้ มีปุ่มก่อนหน้า/ถัดไป จุดบอกฉาก
- 🎤 ไฮไลต์คำตามเสียงอ่าน (read-along) ช่วยเด็กหัดอ่าน
- 🪄 โหมดเล่าต่อเองทุกฉาก + คอนเฟตตีฉลองตอนจบ
- 🔁 ถ้าโหลดไฟล์เสียงไม่ได้ จะสลับไปใช้เสียงในเครื่อง (Web Speech API) อัตโนมัติ

## 🚀 วิธีเปิด
เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลย (แนะนำ Safari หรือ Chrome เพื่อให้มีเสียงภาษาไทย)

หรือรันผ่าน local server:
```bash
python3 -m http.server 4173
# แล้วเปิด http://localhost:4173
```

## 🛠️ เทคโนโลยี
HTML/CSS/JavaScript ล้วน ไม่มี dependency ตอนรัน — เสียงเป็นไฟล์ MP3 ในโฟลเดอร์ `audio/`, ภาพเป็น SVG วาดในโค้ด

## 🔊 สร้างไฟล์เสียงใหม่ (ถ้าแก้ข้อความนิทาน)
ไฟล์เสียงถูกสร้างล่วงหน้าด้วย [edge-tts](https://github.com/rany2/edge-tts) เพราะ Edge TTS เรียกจากเบราว์เซอร์โดยตรงไม่ได้ (ถูกบล็อกด้วย origin)

```bash
python3 -m venv ttsenv && ./ttsenv/bin/pip install edge-tts
./ttsenv/bin/python tools/gen_audio.py          # สร้างใหม่ทุกเรื่อง
./ttsenv/bin/python tools/gen_audio.py fox-grapes  # เฉพาะเรื่องเดียว
```
สคริปต์จะอ่านข้อความจาก `index.html` แล้วเขียนไฟล์ `audio/{story-id}-{ฉาก}.mp3`
