# 🌈 นิทานหรรษา — เว็บเล่านิทานสำหรับเด็ก

เว็บเล่านิทานที่ใช้ **เสียง AI ภาษาไทย** เล่าเรื่อง พร้อม **ภาพประกอบ** และให้เด็ก **แตะแต่ละฉากเพื่อฟัง** ได้เอง

## ✨ ฟีเจอร์
- 📖 นิทาน 9 เรื่อง พร้อมข้อคิดสอนใจ (กระต่ายกับเต่า, ลูกเป็ดขี้เหร่, ราชสีห์กับหนูน้อย, ลูกหมูสามตัว, เด็กเลี้ยงแกะ, มดกับตั๊กแตน, หมาจิ้งจอกกับองุ่น, กากับเหยือกน้ำ, สุนัขกับเงาในน้ำ)
- 🎧 เสียง AI เล่าภาษาไทย ผ่าน Web Speech API (ทำงานออฟไลน์ ไม่ต้องใช้ API key)
- 🎨 ภาพประกอบทุกฉากเป็น SVG วาดในโค้ด คมชัดทุกขนาดจอ
- 👆 เด็กแตะรูป/ปุ่มเพื่อฟัง มีปุ่มก่อนหน้า/ถัดไป จุดบอกฉาก
- 🎤 ไฮไลต์คำตามเสียงอ่าน (read-along) ช่วยเด็กหัดอ่าน
- 🪄 โหมดเล่าต่อเองทุกฉาก + คอนเฟตตีฉลองตอนจบ

## 🚀 วิธีเปิด
เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลย (แนะนำ Safari หรือ Chrome เพื่อให้มีเสียงภาษาไทย)

หรือรันผ่าน local server:
```bash
python3 -m http.server 4173
# แล้วเปิด http://localhost:4173
```

## 🛠️ เทคโนโลยี
HTML/CSS/JavaScript ล้วน ไฟล์เดียวจบ ไม่มี dependency — Web Speech API สำหรับเสียง, SVG สำหรับภาพ
